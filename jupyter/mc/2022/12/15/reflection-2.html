Unit 6: Primitives

See Jupyter Notebook here

Takeaways:

  • Arrays are a collection of data of the same type
  • Arrays are indexed starting at 0
  • Arrays are immutable
  • Need to import java.util.Arrays to use Arrays.toString()
  • Access elements through array[index]

Examples:

int[] arr = {1, 2, 3, 4, 5};
System.out.println(Arrays.toString(arr));

Unit 7: ArrayList

See Jupyter Notebook here

Takeaways:

  • ArrayList is a resizable array
  • Primitive data types: int, double, boolean, char
  • Wrapper classes: Integer, Double, Boolean, Character
  • ArrayLists are mutable
  • Need to import java.util.ArrayList to use ArrayList methods

Examples:

ArrayList<Integer> arr = new ArrayList<Integer>();
arr.add(1);
arr.add(2);
System.out.println(arr);

Unit 8: 2D Arrays

See Jupyter Notebook here

Takeaways:

  • 2D Arrays are arrays of arrays
  • 2D Arrays are indexed starting at [0][0] Array = a data structure used to implement a collection (list) of primitive or object reference data Element = a single value in the array Index = the position of the element in the array (starts from 0) Array Length = the number of elements in the array Is public, so can be accessed in any class Is also final, so can’t change it after array has been created

Unit 9: Inheritance

See Jupyter Notebook here

Takeaways:

  • Inheritance is a way to reuse code
  • Subclass inherits all the methods and variables from the superclass
  • Subclass can override methods from the superclass
  • Subclass can add new methods and variables
  • PolyMorphism: the ability to call the same method on different objects and have different behaviors
  • Superclass: the class being inherited from

Unit 10: Recursion

Our group presentation

Takeaways:

  • Recursion is a method that calls itself
  • Base case is the condition that stops the recursion
  • Recursive case is the condition that calls the method again
  • Recursive methods must have a base case
